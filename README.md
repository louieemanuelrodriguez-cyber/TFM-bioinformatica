# TFM Bioinformática

Este repositorio contiene el código utilizado para el análisis de datos clínicos y bioquímicos
del Trabajo Fin de Máster en Bioinformática.

## Contenido
- Limpieza y preparación de datos
- Análisis descriptivo
- Análisis de componentes principales (PCA)
- Modelos de aprendizaje automático
- Cálculo de score metabólico

## Reproducibilidad
El código está organizado en scripts independientes para facilitar su revisión.
